Media Help

With the media module, you can manage files and folder right in the baseweb admin. It's a convenient way to add or remove files that your website can link to or show as images.

In the view to the left you see your current folders and files of the configured root folder for media. The root must reside in your web folder and must be read and writable by the server.

You can browse the folders and create new ones, or if there are empty folder, remove them.

Selecting a folder allows you to upload images or other files into it. You can also remove any uploaded file by selecting it and deleting it.
