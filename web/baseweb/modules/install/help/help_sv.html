Hjälp för Installera

Här installerar du en helt ny baseweb på din server. Detta verktyg är främst till för att användas under utveckling av din webbsida. Varje gång du väljer att installera baseweb så raderas föregående installation helt.

Om du vill kan du lägga till lite data för test i din installation. Markera bara kryssrutan "Lägg till testdata" innan du installerar.
